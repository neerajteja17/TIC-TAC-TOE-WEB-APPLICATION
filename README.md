# 🎨 Tic Tac Toe - Cartoon Fun

A simple and stylish **Tic Tac Toe** web game built using **HTML**, **CSS**, and **JavaScript** with animated popups and a fun cartoon theme. This version features a clean popup without images.

---

## 🚀 Live Demo
https://neerajteja17.github.io/TIC-TAC-TOE-WEB-APPLICATION/

---

## 📋 Features

- Classic **Tic Tac Toe (X vs O)** game
- Fun **cartoon-style** interface
- **Popup message** for win or draw
- **Restart Game** button
- Fully **responsive** for all devices

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript**
